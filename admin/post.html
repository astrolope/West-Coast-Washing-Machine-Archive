Instructions

Jekyll being a static site generator cannot actually use a form to input pages without intense amounts of scripting. This form is a simple web-app designed to help you generate the necessary front-matter and mark-up to make your posts read properly.
To create a post first create a new file in the "_posts" directory titled "2014-05-04-post-title.md" respectively named to the days date and title of the post.
When that's done copy and paste the formatted front-matter created in this form and save the post file you've created.
The only markup editor in the nav bar to the left creates a nice interface for seeing how your posts will look once on the site.
Congratulations! You've created a Jekyll post, now close this alert and get posting!
